# SistemKantin
Sistem Manajemen Kantin adalah aplikasi basis data yang dirancang untuk mengelola operasional kantin secara efisien. Sistem ini mencakup manajemen menu, pemesanan, meja, dan laporan penjualan. Dibangun menggunakan MySQL dengan struktur database yang terencana dan siap untuk dikembangkan menjadi aplikasi berbasis web atau desktop
